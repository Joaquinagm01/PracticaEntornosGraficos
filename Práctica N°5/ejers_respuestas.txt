Función mail()

bool mail ( string $to , string $subject , string $message [, string $additional_headers [, string $additional_parameters ]] )

Ejercicio 1

Script en PHP para enviar un correo electrónico con formato HTML a través del servidor:

<?php
$destinatario = "destino@ejemplo.com";
$asunto = "Correo de prueba";

$mensaje = '
<html>
<head>
  <title>Correo HTML</title>
</head>
<body>
  <h1>Mensaje desde PHP</h1>
  <p>Esto es una prueba de envío de correo en formato HTML.</p>
</body>
</html>';

$headers = "MIME-Version: 1.0\r\n";
$headers .= "Content-type: text/html; charset=UTF-8\r\n";
$headers .= "From: webmaster@tu-sitio.com\r\n";
$headers .= "Reply-To: webmaster@tu-sitio.com\r\n";

mail($destinatario, $asunto, $mensaje, $headers);
?>

Ejercicio 2

Página de Contacto con un formulario para que los visitantes puedan enviar consultas al webmaster:

<?php
if ($_SERVER["REQUEST_METHOD"] === "POST") {
    $nombre = $_POST["nombre"] ?? "";
    $email = $_POST["email"] ?? "";
    $consulta = $_POST["consulta"] ?? "";

    $destinatario = "webmaster@tu-sitio.com";
    $asunto = "Consulta desde la página de contacto";
    $mensaje = "Nombre: $nombre\nEmail: $email\n\nConsulta:\n$consulta";
    $headers = "From: $email\r\n";
    $headers .= "Reply-To: $email\r\n";

    mail($destinatario, $asunto, $mensaje, $headers);
    echo "Consulta enviada correctamente";
}
?>

<form method="post" action="">
  <label>Nombre:</label>
  <input type="text" name="nombre">
  <br>
  <label>Email:</label>
  <input type="email" name="email">
  <br>
  <label>Consulta:</label>
  <textarea name="consulta" rows="5" cols="30"></textarea>
  <br>
  <button type="submit">Enviar</button>
</form>

Ejercicio 3

Script para que un visitante recomiende el sitio a un amigo:

<?php
if ($_SERVER["REQUEST_METHOD"] === "POST") {
    $amigo = $_POST["amigo"] ?? "";
    $email_amigo = $_POST["email_amigo"] ?? "";
    $nombre = $_POST["nombre"] ?? "";

    $destinatario = $email_amigo;
    $asunto = "$nombre te recomienda este sitio";
    $mensaje = '<html><body>';
    $mensaje .= '<p>Hola ' . htmlspecialchars($amigo) . ',</p>';
    $mensaje .= '<p>' . htmlspecialchars($nombre) . ' te recomienda visitar este sitio web.</p>';
    $mensaje .= '<p><a href="https://tu-sitio.com">Ir al sitio</a></p>';
    $mensaje .= '</body></html>';

    $headers = "MIME-Version: 1.0\r\n";
    $headers .= "Content-type: text/html; charset=UTF-8\r\n";
    $headers .= "From: webmaster@tu-sitio.com\r\n";

    mail($destinatario, $asunto, $mensaje, $headers);
    echo "Recomendación enviada correctamente";
}
?>

<form method="post" action="">
  <label>Tu nombre:</label>
  <input type="text" name="nombre">
  <br>
  <label>Nombre del amigo:</label>
  <input type="text" name="amigo">
  <br>
  <label>Email del amigo:</label>
  <input type="email" name="email_amigo">
  <br>
  <button type="submit">Enviar recomendación</button>
</form>

Variables de sesión

Ejercicio 4

Script para contar las páginas visitadas por un usuario durante su sesión:

<?php
session_start();

if (!isset($_SESSION["visitas"])) {
    $_SESSION["visitas"] = 1;
} else {
    $_SESSION["visitas"]++;
}

echo "Cantidad de páginas visitadas en esta sesión: " . $_SESSION["visitas"];
?>

Si se quiere llevar el conteo de distintas páginas, se puede guardar una variable por cada página visitada dentro de $_SESSION o incrementar el contador en cada carga de la página.
